# Dora
방치형 수집 도감 등록 1인 개발 게임입니다.<br>
* 조작 방법
  * 이동 : WASD
  * 대화 : F
  * 설정 : ESC

## 다운로드
[최신 버전 다운로드]<br>
https://github.com/yhaaan/Dora/releases/latest

## 1.0.4 패치노트

* 골드 소모 및 획득 시 표시되는 골드량이 더 빠르게 반영됩니다
* 게임을 접속 했을 때 요정의 아이템이 Full 일시 남은시간이 현재 레벨의 획득 시간이 아닌 무작위로 설정 되는 버그 수정
